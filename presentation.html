The Problem

Claude Code is powerful, but stuck on your terminal

Imagine this scenario — watch what happens when you step away:

💻
You start a task
"Refactor the auth module" — Claude starts working in your terminal
→
☕
You step away
Coffee break, meeting, or just stretching your legs
→
❓
Claude asks a question
"Should I use JWT or sessions?" — blocked, waiting for your answer
→
⏳
Minutes pass...
Claude sits idle. No notification. No progress. Momentum lost.
→
😱
You return
"Oh, it was waiting for me this whole time..."

Without Claude Code Anywhere

  • Claude Code only works in your terminal
  • Step away = Claude sits idle waiting for you
  • No mobile access to coding sessions
  • Permission prompts block until you return
  • No way to know Claude needs you
  • Context lost when terminal disconnects

With Claude Code Anywhere

  • Access Claude from any device with a browser
  • Teams notification the instant Claude needs you
  • Respond from your phone during coffee break
  • Sessions persist across server restarts
  • QR code to instantly open on mobile
  • Import terminal sessions into web UI
Architecture

How It Works

Click any node to see details. Watch the data flow in real-time.

📱
Your Device
Phone / Laptop
Open the web UI from any device. Scan the QR code from your terminal or type the public URL. Works on phones, tablets, and laptops.
🔐
Tunnel
devtunnel
Microsoft devtunnel creates a public HTTPS URL for the server. Handles auth and provides the QR code for instant phone access.
⚙️
Core
Node.js Server
Pure Node.js HTTP server. Manages sessions, spawns Claude CLI processes, streams output, handles webhooks. Zero frameworks — only 2 npm dependencies.
🤖
AI Engine
Claude CLI
Claude Code CLI runs as a child process with --output-format json. Each session gets its own persistent conversation via --resume. Stdin disabled for safety.
📁
Workspace
Your Code
Each chat has its own working directory. Claude reads, edits, and creates files in your project. Change directory anytime with /cwd.
💬
Teams
@ClaudeCode
@mention Claude in any Teams channel. Messages go through the Outgoing Webhook to Cloudflare, then to the server. Responses post back to the channel.
☁️
Tunnel
Cloudflare
Cloudflare quick tunnel for Teams webhooks. Unlike devtunnel, it has no interstitial page — critical for server-to-server webhook calls from Teams.
🔔
Hooks
Notify Teams
Claude CLI hooks detect Stop, Permission Prompt, and Elicitation events. Sends an Adaptive Card to Teams so you know when Claude needs you — even from the terminal.
📴
Alerts
Push / Toast / Email
Multi-channel notifications: Windows desktop toast, browser Web Push, Outlook email, and in-page toast with sound. At least one will reach you.

CLI Notification Hooks

Even from the terminal, get Teams alerts when Claude needs you:

Stop Event

"Claude is waiting for your response" — finished and needs your next input.

Permission Prompt

"Claude needs permission" — wants to run a command or modify a file.

Elicitation Dialog

"Claude needs your decision" — asking you to choose between options.

Getting Started

Setup in Under 2 Minutes

Three ways to get up and running. Pick what suits you.

Recommended

Claude CLI Setup

claude
# type: /setup

Claude walks you through everything interactively. Installs prerequisites, configures Teams, sets up hooks, starts the server.

Setup Script

# Windows
.\setup.ps1

# macOS / Linux
./setup.sh

Platform-specific script checks prerequisites, installs missing tools, and starts the server.

Manual Setup

npm install
npm start

Install Node.js 18+, Claude CLI, and devtunnel yourself. Then npm install and go.

Prerequisites

ToolRequired?Purpose
Node.js v18+ Required Runs the server
Claude Code CLI Required The AI engine — needs Anthropic API key or Max subscription
devtunnel Optional Remote access from phone/tablet via public HTTPS URL
cloudflared Optional Teams webhook tunnel (no interstitial page)

Motive

AI coding assistants like Claude Code are transforming how developers work — but they're locked to the terminal. We built Claude Code Anywhere to break that barrier: access your AI assistant from any device, stay notified through Teams, and never lose momentum when you step away from your desk. Our goal is to make AI-assisted development truly mobile and collaborative.

Future Scope

Interactive Teams Action Cards
When Claude asks a question (Yes/No, A/B/C), send Adaptive Cards with clickable buttons to Teams.
Multi-User Collaboration

File Browser & Editor
Browse project files, view diffs, and approve changes directly from the web UI.
Scheduled Tasks
